Detailed Description

3D Phong model lighting class

A class which provides a wide range of shaders for different circumstances - utilkising a basic phong shader model. For more advanced features such as normal mapping and specular mapping models with UV mapped textures can be uesd with SO_AssimpModel. The functionaility required is selected by use of the SO_ShaderOptions enums.
