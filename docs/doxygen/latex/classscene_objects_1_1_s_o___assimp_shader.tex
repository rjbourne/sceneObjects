\doxysubsection{Detailed Description}
The shader for an \mbox{\hyperlink{classscene_objects_1_1_s_o___assimp_mesh}{S\+O\+\_\+\+Assimp\+Mesh}} object. 

This shader allows the use of diffuse, specular and normal maps if textures are provided It is intended to be used with the \mbox{\hyperlink{classscene_objects_1_1_s_o___assimp_mesh}{S\+O\+\_\+\+Assimp\+Mesh}} class -\/ although this is not a requirement
