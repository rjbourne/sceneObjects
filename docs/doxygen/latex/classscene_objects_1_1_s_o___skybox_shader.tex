\doxysubsection{Detailed Description}
Generate a skybox using 6 images which will appear in the background. 

Generate a skybox using a vector of 6 images in order right (+x), left (-\/x), top (+y), bottom (-\/y), front (-\/z), back (+z) max size 2048x2048 (some frames delay in render function if larger)
