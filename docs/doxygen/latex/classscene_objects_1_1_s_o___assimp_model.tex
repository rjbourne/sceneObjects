\doxysubsection{Detailed Description}
A class which can load in object files using the assimp library. 

This class can load in object files from many formats using the assimp library. It currently supports loading diffuse, normal and specular textures as well as the mesh. It supports multiple-\/mesh files, storing each as an individual \mbox{\hyperlink{classscene_objects_1_1_s_o___assimp_mesh}{S\+O\+\_\+\+Assimp\+Mesh}}
