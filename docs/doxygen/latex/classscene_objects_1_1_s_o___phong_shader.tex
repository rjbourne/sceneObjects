\doxysubsection{Detailed Description}
3D Phong model lighting class 

A class which provides a wide range of shaders for different circumstances -\/ utilkising a basic phong shader model. For more advanced features such as normal mapping and specular mapping models with UV mapped textures can be uesd with \mbox{\hyperlink{classscene_objects_1_1_s_o___assimp_model}{S\+O\+\_\+\+Assimp\+Model}}. The functionaility required is selected by use of the S\+O\+\_\+\+Shader\+Options enums.
