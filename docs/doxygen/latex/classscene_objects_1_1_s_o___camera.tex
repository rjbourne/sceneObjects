\doxysubsection{Detailed Description}
A 3D camera capable of creating view/projection matrices to linked shader programs on call. 

This camera class is designed to allow multiple objects to be kept in the same scene by automatically updating the relevant view and projection matrices with the \mbox{\hyperlink{classscene_objects_1_1_s_o___camera_ac7e19d1848b7b606602eb0787e41e8d4}{update\+View\+Matrix()}} and \mbox{\hyperlink{classscene_objects_1_1_s_o___camera_a564ca4e151b155ebc946860dfd4309d0}{update\+Projection\+Matrix()}} methods. Any derived class of \mbox{\hyperlink{classscene_objects_1_1_s_o___shader}{S\+O\+\_\+\+Shader}} can be added to the camera.
