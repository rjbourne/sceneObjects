\doxysubsection{Detailed Description}
struct containing data for a simple mesh which can be used or manipulated by the user 

struct consists of a vector of vec3s called {\ttfamily vertices} which are the vertices of the mesh, and a vector of ints {\ttfamily face\+Elements} which give the vertices in groups of three to be rendered as triangles by Open\+GL
